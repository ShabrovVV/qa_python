# BooksCollector

Сервис для управления коллекцией книг. Позволяет добавлять книги, устанавливать жанры, фильтровать книги для детей, добавлять книги в избранное и удалять их оттуда.

---

## Тестирование

Проект покрыт автотестами на базе `pytest`. Реализован **31 тест**, покрывающий все методы класса `BooksCollector`.

### Покрытие тестами

| Метод | Тестов | Что проверяется |
|---|---|---|
| `add_new_book` | 5 | Добавление книг, ограничение 40 символов, пустое имя, дубликаты |
| `set_book_genre` | 3 | Установка жанра, книга не существует, невалидный жанр |
| `get_books_with_specific_genre` | 3 | Поиск по жанру, пустой список, невалидный жанр |
| `get_book_genre` | 4 | Книга есть/нет, жанр установлен/не установлен, регистр |
| `get_books_for_children` | 3 | Фильтрация по возрастному рейтингу |
| `add_book_in_favorites` | 3 | Добавление, книга не в словаре, дубликат |
| `delete_book_from_favorites` | 2 | Удаление, удаление несуществующей книги |
| `get_list_of_favorites_books` | 2 | Пустой список, список с книгами |

---

## Запуск тестов

Все тесты:
```bash
pytest -v tests.py
```

Оценка покрытия:
```bash
pytest --cov=main tests.py -v
```

HTML-отчёт по покрытию:
```bash
pytest --cov=main --cov-branch --cov-report=html tests.py
```
